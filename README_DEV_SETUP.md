# Maksy Developer Setup

## Prerequisites
- Node 20+
- pnpm 8+
- Supabase CLI (optional)

## Run Locally
```bash
pnpm install
cd apps/web
pnpm dev
```

## Project Structure
- apps/web – Next.js app
- .github/workflows – CI pipelines
- supabase/ – SQL & policies

## Environments
- .env.local – local secrets
- .env.example – sample placeholders
